Laravel Passport Setup Commands
==================================

Run these commands in order to complete the Laravel Passport setup:

1. Install Passport dependencies:
   composer install

2. Clear any cached configuration:
   php artisan config:clear
   php artisan route:clear

3. Run database migrations for Passport:
   php artisan migrate

4. Install Passport (creates encryption keys):
   php artisan passport:install --force

5. Publish Passport's service provider (optional):
   php artisan vendor:publish --provider="Laravel\Passport\PassportServiceProvider"

6. Cache optimized configuration:
   php artisan config:cache

7. Test the setup:
   php artisan route:list | grep auth

Important Notes:
- Make sure your database is configured and running
- The --force flag on passport:install will overwrite existing keys
- After setup, test authentication endpoints with your frontend
- Check that HTTP-only cookies are being set correctly

API Endpoints Available:
- POST /api/auth/register
- POST /api/auth/login  
- POST /api/auth/logout (requires auth)
- GET  /api/auth/me (requires auth)
- POST /api/auth/refresh (requires auth)
- PUT  /api/auth/profile (requires auth)
